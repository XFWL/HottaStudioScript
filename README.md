# 异环店长特供工具

# **你得有 娜娜莉 和 白藏 这个脚本才有意义！！！**

![](https://raw.githubusercontent.com/XFWL/HottaStudioScript/refs/heads/main/tip/tip3.png)

## 功能说明

这是一个由 Trae CN （AI）生成的带GUI界面的自动化工具，用于：

- 选择目标窗口（支持全屏游戏和窗口模式）
- 识别指定图像并执行相应操作：
  1. 识别 店长特供 图片，执行F键操作
  2. 识别 开始营业 图片，执行点击操作
  3. 识别 锤子 图片，执行持续点击操作
  4. 识别 领取 图片，执行点击操作
- 支持循环执行和重试功能
- 全局取消热键（F12），如果是一直执行，它会在每轮结束取消整个循环（本人python水平有限，Trae CN不知道为什么只能这样处理）

## 使用方法

1. **准备图像文件**：（自带了，用不了再说）

   - 在脚本同一目录下创建 `1080` 和 `2k` 文件夹
   - 将对应分辨率的图像放入相应文件夹：
     - `image1.png`：店长特供图片
     - `image2.png`：开始营业图片
     - `image3.png`：锤子图片
     - `image4.png`：领取图片

2. **注意事项**：

   1. 确保默认选择 `店长特供`

      ![](https://raw.githubusercontent.com/XFWL/HottaStudioScript/refs/heads/main/tip/tip1.png)

   2. 确保默认选中 2-8 关卡，没选中可以先进去玩一局

      ![](https://raw.githubusercontent.com/XFWL/HottaStudioScript/refs/heads/main/tip/tip2.png)

3. **运行工具**：

   - 以管理员权限运行 `image_recognition_automation.exe`
   - 或从源代码运行：`python image_recognition_automation.py`

4. **GUI界面设置**：

   - **选择目标窗口**：从下拉列表中选择需要操作的窗口
   - **选择分辨率**：根据当前游戏窗口分辨率选择
   - **操作设置**：
     - 第一步：设置F键次数
     - 第二步：设置点击次数
     - 第三步：设置延迟时间和持续点击时间
     - 第四步：设置点击次数和延迟时间
   - **循环设置**：选择执行次数（一次、指定次数、一直执行）
   - **执行选项**：选择要执行的操作（单个操作或所有步骤）
   - **重试功能**：启用在错误时重试当前操作

5. **开始执行**：
   - 点击"开始执行"按钮
   - 执行过程中可以按F12键取消操作，如果是一直执行，它会在每轮结束取消整个循环（本人python水平有限，Trae CN不知道为什么只能这样处理）

## 注意事项

- **必须以管理员权限运行**：否则可能无法操作游戏窗口
- **确保目标窗口可见**：窗口不能最小化或被其他窗口遮挡
- **图像识别要求**：
  - 图像文件必须清晰可见
  - 确保图像在窗口中的位置和大小与截图一致
  - 不同分辨率需要使用对应文件夹中的图像
- **取消操作**：
  - 按下F12键可以立即取消正在执行的操作
  - 取消后程序会退出当前执行流程
- **性能影响**：
  - 持续点击操作可能会占用较多系统资源
  - 建议在执行长时间操作时不要运行其他占用资源的程序
- **安全提示**：
  - 本工具仅用于自动化操作，请勿用于恶意用途
  - 执行操作时请确保游戏账号安全

## 故障排除

- **键盘操作无反应**：
  - 确保以管理员权限运行
  - 检查目标窗口是否在前台
  - 尝试重新启动工具

- **窗口列表为空**：
  - 点击"刷新窗口列表"按钮
  - 确保目标窗口已打开

- **图像识别失败**：
  - 确保图像文件存在且清晰
  - 检查图像路径是否正确
  - 尝试调整图像或提高匹配阈值

- **取消热键不工作**：
  - 确保工具以管理员权限运行
  - 检查F12键是否被其他程序占用

- **持续点击操作异常**：
  - 检查持续时间设置是否合理
  - 确保目标窗口在持续点击期间保持可见
